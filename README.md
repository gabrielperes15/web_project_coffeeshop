# Triple Espresso

## Descrição

O Triple Espresso é uma página web para uma cafeteria especial localizada na biblioteca Triple Peaks. A página mostra informações sobre o café, receitas de preparação em vídeo, um formulário de reserva de mesas e links para redes sociais.

O projeto foi desenvolvido utilizando HTML5 e CSS3, com organização de estilos seguindo a metodologia BEM (Block Element Modifier) usando uma estrutura de arquivos Flat BEM.

## Tecnologias utilizadas

- HTML5
- CSS3
- Metodologia BEM
- Google Fonts (Inter e Noto Serif)

## Planos de melhoria

- **Sistema de reservas funcional:** implementar um backend para que as reservas feitas pelo formulário sejam salvas em um banco de dados e o usuário receba uma confirmação por e-mail.

- **Modo escuro:** adicionar um botão que permita ao usuário alternar entre o tema claro e o tema escuro da página, mas talvez isto seja possivel apenas com JavaScript.
